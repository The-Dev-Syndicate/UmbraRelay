## Quick Reference

- **Adding RSS Feed**: Sources → RSS Feed tab → Fill form → Add
- **Adding GitHub Source**: Sources → GitHub tab → Fill form (with token) → Add
- **Mark Item Read**: Click item → View details
- **Archive Item**: Click item → Archive button
- **Sync Source**: Sources → Click "Sync" button
- **Filter Inbox**: Use filter buttons (All/Unread/Read/Archived)
